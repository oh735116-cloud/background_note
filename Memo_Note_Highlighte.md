# 메모 기반 자동 하이라이터 개발 기획서

## 1. 프로젝트 개요

**프로젝트명:** Memo-Driven Highlighter  
**형태:** Chrome Extension Manifest V3  
**개발 환경:** VS Code, HTML, CSS, Vanilla JavaScript  
**목표:** 사용자가 저장한 키워드와 메모를 기반으로 웹페이지의 관련 텍스트를 자동 하이라이트하고, 마우스 hover 시 메모를 확인할 수 있는 학습 보조 확장 프로그램을 만든다.

이 확장 프로그램은 웹에서 공부하거나 자료를 읽을 때 반복해서 확인해야 하는 단어, 개념, 문장을 개인 메모와 연결해준다. 사용자는 키워드와 메모를 저장하고, 이후 다른 웹페이지에서도 같은 키워드가 나타나면 자동으로 강조 표시를 확인할 수 있다.

## 2. 핵심 사용자 가치

- 저장한 키워드가 웹페이지에서 자동으로 강조되어 중요한 내용을 놓치지 않는다.
- 하이라이트에 마우스를 올리면 연결된 메모를 바로 확인할 수 있다.
- 팝업에서는 빠르게 메모를 저장하고, 사이드패널에서는 화면을 닫지 않고 계속 관리할 수 있다.
- 데이터는 `chrome.storage.local`에 저장되어 브라우저 안에서 간단하게 유지된다.

## 3. 주요 기능

### 3.1 메모 저장 및 관리

- 키워드, 메모, 색상, 사용 여부를 저장한다.
- 저장된 항목은 목록에서 수정, 중지/재개, 삭제할 수 있다.
- 같은 키워드를 다시 저장하면 기존 항목을 갱신한다.
- 우클릭 컨텍스트 메뉴로 선택한 텍스트를 임시 키워드로 가져올 수 있다.

### 3.2 자동 하이라이트

- `content.js`가 웹페이지 본문 텍스트를 탐색한다.
- 저장된 활성 키워드를 찾아 `<span class="mdh-highlight">`로 감싼다.
- 긴 키워드를 우선 매칭하여 짧은 키워드가 긴 키워드 일부를 먼저 차지하지 않도록 한다.
- 스크립트, 스타일, 입력창 등 하이라이트하면 안 되는 영역은 제외한다.

### 3.3 메모 툴팁

- 하이라이트된 텍스트에 마우스를 올리면 저장된 메모가 툴팁으로 표시된다.
- 메모가 없는 항목은 툴팁을 표시하지 않는다.
- 하이라이트 배경색에 따라 글자색을 자동으로 흰색 또는 어두운 색으로 바꿔 가독성을 높인다.

### 3.4 팝업과 사이드패널 UI

- 팝업은 빠른 저장과 최근 메모 확인에 사용한다.
- 사이드패널은 웹페이지를 보면서 계속 열어둘 수 있는 관리 화면으로 사용한다.
- Chrome 확장 팝업은 웹페이지를 클릭하면 닫히는 정책이 있으므로, 지속 작업은 사이드패널을 통해 해결한다.
- UI는 다이어리 감성의 부드러운 색상, 둥근 카드, 캐주얼한 폰트 스타일을 적용한다.

## 4. 기술 스택

- **Platform:** Chrome Extension Manifest V3
- **Language:** HTML5, CSS3, Vanilla JavaScript
- **Chrome APIs:** `chrome.storage`, `chrome.contextMenus`, `chrome.sidePanel`, `chrome.tabs`
- **DOM 처리:** `TreeWalker`, 텍스트 노드 탐색, 동적 `<span>` 삽입

## 5. 파일 구성

| 파일                         | 역할                                                           |
| ---------------------------- | -------------------------------------------------------------- |
| `manifest.json`              | 확장 프로그램 권한, 백그라운드, content script, sidepanel 설정 |
| `Memo-Driven_Highlighter.js` | 컨텍스트 메뉴, 초기 저장소 구조 보장, 사이드패널 동작 설정     |
| `content.js`                 | 웹페이지 자동 하이라이트, hover 메모 툴팁, 색상 대비 처리      |
| `sidepanel.html`             | 지속적으로 열어두는 메모 관리 화면                             |
| `ui.js`                      | 메모 CRUD, storage 연동, 탭 새로고침 메시지 처리               |
| `ui.css`                     | 다이어리 감성 UI 스타일                                        |

## 6. 구현 흐름

1. 사용자가 팝업 또는 사이드패널에서 키워드와 메모를 저장한다.
2. `ui.js`가 데이터를 `chrome.storage.local`의 `highlights` 배열에 저장한다.
3. 저장 후 열려 있는 웹페이지 탭에 `MDH_REFRESH_HIGHLIGHTS` 메시지를 보낸다.
4. `content.js`가 저장된 키워드를 다시 읽고 본문에 하이라이트를 적용한다.
5. 사용자가 하이라이트 위에 마우스를 올리면 메모 툴팁을 표시한다.

## 9. 기대 효과

이 프로젝트는 단순한 북마크나 메모장을 넘어, 사용자의 지식과 웹페이지 내용을 직접 연결한다. 공부 중 반복해서 만나는 개념을 자동으로 강조하고, 개인 메모를 즉시 확인할 수 있어 학습 흐름을 끊지 않는 보조 도구로 활용할 수 있다.
